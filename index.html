FAQ

Common questions.

Straight answers, no filler.

Is the script bannable?

No, the script isn't bannable. We make sure Proton Hub stays up to date so it remains undetected across patches.

Does the script cause lag?

It depends. If you're running a thousand features at once, yes. But if you use our built-in anti-afk and anti-lag system, it will actively try to reduce that for you.

Is the script keyless?

Yes, fully keyless right now. All scripts will stop being keyless starting from version 3.5. The current version of Proton Hub is V2, so you're good for a while.
